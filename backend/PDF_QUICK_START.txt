🔧 INSTALLATION (Windows):

   Step 1: Install Poppler
   ----------------------
   Option A - Automated (Recommended):
   → Open PowerShell as Administrator
   → cd backend
   → .\install-poppler-windows.ps1

   Option B - Using Chocolatey:
   → choco install poppler

   Option C - Manual:
   → Download: https://github.com/oschwartz10612/poppler-windows/releases/
   → Extract to: C:\Program Files\poppler
   → Add to PATH: C:\Program Files\poppler\Library\bin
   → Restart terminal

   Step 2: Install npm package
   ---------------------------
   → cd backend
   → npm install

   Step 3: Verify installation
   ---------------------------
   → pdftoppm -v
   → node test-pdf-conversion.js

🐛 TROUBLESHOOTING:
   Error: "spawn pdftoppm ENOENT"
   → Poppler not installed or not in PATH
   → Run: pdftoppm -v to verify
   → Restart terminal after installation

   Error: "PDF conversion failed"
   → Invalid PDF file
   → Try with different PDF

   Error: "EACCES: permission denied"
   → backend/temp/ directory not writable
   → Check folder permissions
